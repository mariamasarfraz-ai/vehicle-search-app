# Vehicle Search App

A client-side web application that allows users to search and filter a dataset of vehicles.

## Features
- Search by make, model, colour, fuel type, or year
- Filter by make, fuel type, colour, and year range
- Sort results by relevance, year, make, or model
- Dynamic filtering without page reload

## Technologies Used
- HTML
- CSS
- JavaScript (ES6)

## How to Run
Open index.html in a web browser.
